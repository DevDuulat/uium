Что такое Black UI Kit?
Это минималистичная библиотека компонентов, созданная на Tailwind CSS с упором на черно-белую эстетику и простоту интеграции.
Как использовать на GitHub Pages?
Просто залейте ваши HTML файлы в репозиторий и настройте GitHub Actions для сборки статических страниц. Все пути в этом ките относительные.
